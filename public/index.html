Archlinux on Vmware

Feb 17th, 2012 | Comments

Enable networking

append to /etc/rc.conf

eth0="dhcp"
INTERFACE=(eth0)
ROUTES=(!gateway)

run command “/etc/rc.d/network restart”

Enable pacman repo

append to “/etc/pacman.conf”

SigLevel=Optional TrustAll

uncomment the repo in “/etc/pacman.d/mirrorlist”

Upgrade

pacman -Syu
